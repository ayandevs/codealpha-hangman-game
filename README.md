# 🎯 Hangman Game

## 📖 Description

This is a simple text-based Hangman Game developed in Python as part of the CodeAlpha Python Programming Internship. The game randomly selects a superhero name from a predefined list, and the player has to guess the word one letter at a time. The player is allowed a maximum of 6 incorrect attempts to guess the correct word.

## ✨ Features

- Randomly selects a superhero name from a predefined list.<br>
- Allows the user to guess one letter at a time.<br>
- Displays the correctly guessed letters and hides the remaining ones.<br>
- Limits the player to 6 incorrect attempts.<br>
- Prevents duplicate letter guesses.<br>
- Validates user input to accept only a single alphabetic character.<br>
- Displays win or lose messages at the end of the game.
